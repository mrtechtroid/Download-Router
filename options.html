How to Use

Getting Started

Download Router helps you automatically organize downloads based on customizable rules. Each rule can match downloads using up to 4 different criteria:

Rule Matchers

  • MIME Type: Match file types (e.g., image/.* for images, application/pdf for PDFs)
  • File URL: Match the download URL (e.g., .*github\.com.* for GitHub downloads)
  • Referrer URL: Match the page you're downloading from (e.g., .*reddit\.com.*)
  • Filename: Match the file name (e.g., .*\.zip$ for ZIP files)
  • In addition, using =SOMETHING will check if the parameter is exactly equal to SOMETHING
  • Using ~SOMETHING for the filename will check if the file extension is exactly equal to SOMETHING

Regular Expressions

All matchers use case-insensitive regular expressions. Here are some useful patterns:

.*\.pdf$             # Files ending with .pdf
image/.*             # Any image MIME type
.*github\.com.*      # URLs containing github.com
^Screenshot.*        # Files starting with "Screenshot"
~pdf                 # File extension is pdf
=https://example.com # Exact URL is matched.

Folder Paths

Specify relative folder paths like Documents/PDFs or Downloads/Images. The extension will create these folders if they don't exist.

Rule Matching

For a rule to trigger:

  • At least one matcher must be filled in
  • ALL filled matchers must match (AND logic)
  • The first matching rule will be used

About Download Router

Download Router is a Chrome extension that automatically organizes your downloads based on customizable rules.

Privacy

This extension:

  • Only processes download information locally
  • Does not collect or send any data
  • Stores rules in your browser's sync storage
  • Only accesses download and tab information when needed

Permissions

  • Downloads: Required to intercept and redirect downloads
  • Active Tab: Used to get referrer URL information
  • Storage: Stores your rules and settings
